1.4 Interface utilisateur Web

Une interface utilisateur, ou UI en anglais, désigne les éléments a l'écran qui permettent d'interragir avec une page Web.

Cela inclut donc l’ensemble des items interactifs et clicables, comme la barre de navigation du site, les boutons ou les liens.

L'UI est aujourd'hui un composant très important et essentiel des sites Web, car l'ergonomie est aujourd'hui une priorité pour les utilisateurs.
