1.7 Navigateurs les plus utilisés

Aujourd'hui, les navigateurs qui sont globalement les plus populaires sont :

  • Chrome - PC / Mobile
  • Safari - Mobile
  • Firefox - PC / Mobile

Sur mobile, Chrome est le plus utilisé en moyenne, suivi de firefox et safari. Sur PC, c'est chrome qui domine aussi principalement. Ce dernier s'étends donc sur les deux plateformes les plus utilisées du marché.
